*Contract work (work-in-progress)*

Using publicly available ocean current datasets, I built a framework for testing motion planning algorithms applied to autonomous ocean buoys. The project visualizes real buoy trajectories using **Python**, **Pandas**, and **Folium**, then uses **Matplotlib** to create a toy 2D simulation environment where custom motion planning code can be evaluated.

Solo work. Currently work-in-progress; no public links yet.
