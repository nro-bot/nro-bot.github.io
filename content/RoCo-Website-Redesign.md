*Contract work — UCLA LEMUR Lab*

Redesign of the UCLA LEMUR Lab's **RoCo** (Robot Compiler) web application. RoCo lets users compose robots from modular components.

**What I did:** Solo work — significant Flask (Python) / Jinja2 templating refactoring, adding Bootstrap (CSS) to make the site mobile and desktop responsive, minor JavaScript for the homepage image carousel, and bash/Python/OpenSCAD scripting to auto-generate STL thumbnails for display in the carousel.
