*Nancy Ouyang, Laura Zharmukhametova, Jamelle Watson-Daniels — Harvard CS279r PL/HCI Graduate Seminar*

[PaperSignals](https://papersignals.withgoogle.com/) are build-it-yourself printable robots that users can control with their voice. The original system was limited to fixed templates. This project built an open-source browser-based UI in Python/Flask that lets users parameterize and customize existing PaperSignal templates without needing to modify code directly.

I co-wrote the Flask web app and built the physical prototypes.
